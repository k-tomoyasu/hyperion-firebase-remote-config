## Installation
Add to build.gradle
```groovy
dependencies {
	debugImplementation "com.willowtreeapps.hyperion:hyperion-core:$hyperion_version"
    debugImplementation "com.fusuma:hyperion-firebase-remote-config:$plugin_version"
}
```
